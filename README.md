# Field Sales Companion — التطبيق النهائي الكامل

تطبيق مبيعات ميداني يعمل دون اتصال (Offline-First PWA) لممثلي مبيعات FMCG.

## ✅ جميع الإصلاحات المُطبقة

### 1. تخطيط المسار (Route Planning)
- محدد القطاع (Tous / S2 / S3) في شاشة المسار
- عرض الأيام المعينة لكل قطاع من Route Planning
- تصفية العملاء حسب القطاع المختار
- محرر كامل في الإعدادات: إضافة/تعديل/حذف/ترتيب الأيام والقطاعات والأحياء

### 2. تصدير طلبات نهاية اليوم
- 3 أعمدة فقط: Customer Name, Product Name, Quantity
- اسم العميل يظهر مرة واحدة لكل عميل

### 3. تعديل وحذف العملاء
- زر "تعديل العميل" في صفحة تفاصيل العميل
- زر "حذف العميل" مع نافذة تأكيد
- الحذف يزيل العميل + جميع الطلبات والزيارات المرتبطة

### 4. الصفحة الرئيسية المبسطة
- فقط 4 أزرار: ابدأ الجولة، البحث، عميل جديد، طلب جديد
- تم إزالة جميع الأقسام غير الضرورية

### 5. التصدير حسب الفترة في الإعدادات
- قسم "تصدير حسب الفترة" في الإعدادات
- اختيار: يوم / أسبوع / شهر
- تصدير كـ CSV (3 أعمدة)

### 6. إدارة المنتجات
- إضافة/تعديل/حذف المنتجات تعمل بشكل صحيح
- المنتجات المحذوفة لا تعود بعد التحديث
- تعديل المنتج يفتح النموذج بالبيانات مُعبأة

### 7. العملاء
- العملاء لا يختفون بعد التصدير أو التحديث
- إنشاء عميل يربطه بالمندوب الحالي بشكل صحيح

### 8. المنتجات التجريبية
- تمت إزالة جميع المنتجات التجريبية
- الحساب يبدأ بقائمة منتجات فارغة
- المنتجات تُضاف فقط بواسطة المستخدم

### 9. خريطة عميل جديد
- الخريطة تظهر بشكل صحيح في شاشة إنشاء عميل جديد
- زر "موقعي" يعمل بشكل صحيح
- حقل الهاتف اختياري

### 10. إصلاحات إضافية
- إزالة أخطاء runtime
- إصلاح البناء (إزالة مراجع Prisma القديمة)
- توافق GitHub Pages
- يعمل دون اتصال

## التشغيل

```bash
# تثبيت الاعتماديات
bun install

# التطوير
bun run dev

# فحص الكود
bun run lint

# البناء للإنتاج
bun run build
```

## النشر على GitHub Pages

```bash
bun run build
# ارفع محتويات مجلد out/ إلى فرع gh-pages
```

الـ basePath مُعد لـ `/sales-companion-offline` — عدّله في `next.config.ts` إذا كان اسم مستودعك مختلفاً.

## التقنيات

- Next.js 16 (App Router, Static Export)
- React 19 + TypeScript 5
- Tailwind CSS 4 + shadcn/ui
- Dexie.js (IndexedDB)
- Zustand (persist)
- React Leaflet
- Recharts
- Framer Motion
- PWA (Service Worker + Manifest)

## ملاحظة مهمة

```js
// في console المتصفح:
indexedDB.deleteDatabase('SalesCompanionDB');
localStorage.clear();
// ثم أعد تحميل الصفحة
```
